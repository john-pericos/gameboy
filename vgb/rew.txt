        This is freeware GB emulator for Windows.

        - Basic Gameboy hardware
        - Super Gameboy
        - Gameboy color
        - MBC1/MBC2/MBC3/MBC5/MMM01/TAMA5/HuC-1/HuC-3 Memory mappers
        - Stereo sound
        - Save states
        - Key remapping

        This Version

        - Fixed multiple bugs
